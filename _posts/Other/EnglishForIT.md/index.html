Jobs In Information Technology

System administrator

Also this can be called sysadmin for short

In information technology, a System Administrator is a person who supports company’s computing environment and ensures good performance of IT services and support systems.

Database Administrator

Also this can be called DBA for short.

Database Administrators help store and organize company’s data. They protect the data from unauthorized users. Usually they work for organizations with large database systems, such as educational institutions, financial firms etc.

Information Security Analyst

An Information Security Analyst protects a computer network and computer systems of the company. They plan and carry out different security measures, for example they install and use software to stimulate cyber attacks in order to test systems.

Computer Network Architect

Computer Network Architects design, build, and maintain a variety of data communication networks. They typically have a bachelor’s degree in computer science or a related field.

Computer Support Specialist

Also this can be called Technical Support Specialist

A Computer Support Specialist assists users who are having problems with software, computers or peripherals (for example printers or scanners). Those people, via online chat or email, are called Help Desk Technicians.

Software Developer

Also this can be called dev for short

Software Developers design, run and test various computer programs and applications. They usually have a bachelor’s degree in computer science or a related field. They also have strong programming skills.

Quality Assurance Analyst

Also this can be called tester

Quality Assurance Analysts are responsible for creating test plans, executing those plans and managing all activities in the plan to ensure that the program works as expected.

Computer Programmer

A Computer Programmer is a person who creates computer software. They write code using computer languages, such as C++, Java, Python etc. Computer programmers create instructions for computers to generate meaningful output (result).

Technician

IT Technician set up workstations with computers and other devices, ensure functionality of computer hardware (HDD, mouses, keyboards etc.), install and configure appropriate software and functions.

Web Developer

Web Developer build and shape a visitor’s experience at a website. They create webpage layouts (headings, paragraphs etc), website styling (colors and fonts), and page features (animations and pictures). They use languages like HTML or JavaScript to create websites or web applications.

Web Designer

A Web Designer is responsible for creating the design and layout of a website or a webpage. They can work on a brand new website or update and improve an already existing site.

The words to explain what you do on a day to day

words meaning
to deal with to take action to do something, to solve problems, to work with something/someone
deal-dealt-dealt
to maintain to make sure that something stays at the same level, condition or standard

Eg:

  • Our managers deal with all kinds of clients every day. So I can say that we maintain the highest level of service.
words meaning
to carry out to do something to perform
to ensure to make certain; to make sure

Eg:

  • In my previous project I carried out the responsibilities of both the Project Manager and the Team Leader.
  • As a QA specialist I worked with a test environment where I tested may aspects of the platform to ensure that it works as desired.
words meaning
to oversee to watch over and control something to make sure that the work is good or satisfactory; to supervise
to deploy to make a software system (website, app) available for use

Eg:

  • A cloud architect oversees application architecture and deploys it in cloud environments like public cloud, private cloud, private cloud and hybrid cloud.
words meaning
to design to create, draw or construct something
to set up to make a piece of equipment ready for use

Eg:

  • I took a course where I learned how to design and write programs that are easy to maintain.
  • I will set up all the necessary equipment in my home office work remotely on this project.
words meaning
to install to put a new program or piece of software into a computer so that you can use it
to configure to change setting of software on a computer, so that it works in the way you want it to

Eg:

  • I use different plugins and libraries every day, so I often need to install and configure new software. That’s why I read a lot of documentation in English.
words meaning
to protect to keep something safe from harm, damage or loss
to assist to help someone or something

Eg:

words meaning
to provide to give someone something that they want or need
to manage to organize and control the work of a company, organization, or some people

Eg:

  • I am a web designer, so I know how to provide the best user experience for your website visitors
  • As a CEO of a big company I need to manage a lot of creative people
words meaning
to estimate to give a general idea of the cost of work or the time you need to do the work
to execute to perform, to do; to make a computer use a program or carry out an instruction

Eg:

  • Project managers usually estimate new projects by analogy, using previous projects and the past experience
  • Sometimes I need to google my questions, for example: “How to execute the code inside a function in JS”
words meaning
to improve to make something better
to run to start or to use a computer program

Eg:

  • Working on my previous project improved my time management and organizational skills.
  • When an application is developed we have to make sure that it runs on different devices and words perfectly.

Programming terms

words description
Algorithm An algorithm is a step-by-step procedure which is used to solve problems or to do calculations. Algorithms are implemented with code.
Value A value is a piece of data. Every value has a type. Values represent information and we can work with it using code.
Variable A variable is a container for a single value. It is named. Difference values can be assigned to a variable.
Assign To put a value into a variable. The process is called assignment
Integer Integer is a type of value that represents whole numbers. It can be positive, negative, or zero.
Float Float is a type of value that is a number with a fractional part. Short for “floating-point number”.
Double Double is a data type that is a number with decimal points. It is similar to float, but has more precision. Short for “double-precision floating-point number”.
String A String is a type of value that represents text. The term is short for “string (sequence) of characters”.
Boolean A boolean is data type that has two values (that are usually called true and false).
Array An array is a type of value that can contain a sequence of other values.
Function A function is a piece of code that is run only when the function called (executed). Functions can take arguments (parameters). When a function finishes running, returns a certain value.
To call To call a function is to run the code in a function.
Argument (Parameter) An argument (parameter) is a value that is passed into a function when it is called.
Class A class is a type that can be created by the programmer. We need a class to created objects of that class.
Object An object is a value created from a class. Objects usually contain some variables and values inside and have methods attached to them.
Method A method is a function that is attached to an object.
Bug A bug is a mistake in a program
Debug To debug is to investigate the program and fix bugs
Comment A text written around code that is ignored by the computer. It is used for writing extra info about your code to help you (for someone else) understand it later
Comment out To turn a piece of code into a comment with the help of special characters
Compiler A compiler is a program that converts your code into a machine-code so that it can be read and executed by a computer
Error An error is a mistaken in a computer process
Constant A constant is a variable that never changes its value
Crash A program crashes it stops running because of an error
Executable An executable is a program which is ready to be run (executed). Short for executable file/program.
Declare To declare in programming means to say that something exists-usually a variable, a function or a class.
Implement To implement means to write and complete something in code , for example, to implement a function or a class.
Instantiate To instantiate means to create an object form a class.
To iterate / Loop A loop is a piece of code that runs itself may times. It can also be used as a verb - to loop (to iterate).
Read To read means to take some data values from another source - usually from a file or over the internet.
Write To write is to send data values to another destination - usually to a file or over the internet.
Syntax Syntax is the grammatical rules of a programming language. Syntax determined if code is written correctly or incorrectly

退出

命令 作用
:q quit
:q! force quit
:wq write (save) and quit

插入

命令 作用
a append (write after this char)
i insert (write before ths char)
o open a new line below
A append after line
I insert before line
O open a new line above
gi insert at the position last edited
ESC quit the insert mode, go back to the normal mode
ctrl+c quit the insert mode, go back to the normal mode
ctrl+[ quit the insert mode, go back to the normal mode

移动

命令 作用
h move left
j move down
k move up
l move right
w/W move to next word/Word start
e/E move to next word/Word tail
b/B move to previous word/Word start
0 move to the first char of the line
^ move to the first non-blank char of the line
$ move to the last char of the line
g_ move to the last non-blank char of the line
gg move to the start of the file
G move to the tail of the file
ctrl+o move to the previous position
H move to the start of the screen
M move to the middle of the screen
L move to the tail of the screen
ctrl+u page up
ctrl+f page down
zz put the line cursor at to the middle of the screen
  • word 指以非空白符分隔的单词
  • Word 指以空白符分隔的单词

行内搜索移动

命令 作用
f{char} move to the next char we searched
F{char} move to the previous char we searched
t{char} move to the char before the next char we searched
; move to the next char meeting the condition
, move to the previous char meeting the condition

文本对象

对象 含义
w word
s sentence
p paragraph
i decorate inner
a decorate around
i( or i) the chars in () but not include ()
a( or a) the chars in () and include ()
i< or i> the chars in <> but not include <>
a< or a> the chars in <> and include <>
i{ or i} the chars in {} but not include {}
a{ or a} the chars in {} and include {}
i[ or i] the chars in [] but not include []
a[ or a] the chars in [] and include []
i" the chars in “” but not include “”
a" the chars in “” and include “”
i' the chars in ‘’ but not include ‘’
a' the chars in ‘’ and include ‘’

删除

命令 作用
x delete a char
daw delete a word and the space around it
diw delete a word
dw the same function as daw
dd delete current line
dt{char} delete between the cursor and the char
  • x 和 d 可以配合数字使用
  • 可以d 文本对象

修改

命令 作用
r replace a char
s delete a char and into the insert mode
R replace next char untill quit
S delete the whole line and into the insert mode
caw/cw delete current word and into the insert mode
ct{char} delete between the cursor and the char and into the insert mode
  • 可以 c 文本对象

查询

命令 作用
/ search from start
? search from tail
n next maching
N previous maching
* next matching of current word
# previous maching of current word

替换

命令 作用
:[range]s/{pattern}/{string}/[flags]
  • range 表示范围，比如 10, 20 表示 10-20 行，%表示全部
  • s 代表 substitute 替换
  • pattern 是要替换的模式
  • string 是替换后的文本
  • flags 替换位标志
    • g (global) 表示全局范围内执行
    • c (confirm) 表示确认，可以确认或者拒绝修改
    • n (number) 报告匹配到的次数而不替换，可以用来查询匹配次数

保存

命令 作用
:w write (save)

复制粘贴

命令 作用
y copy
p paste
d cut
yy copy the whole line
  • 可以 y 文本对象
  • 寄存器
    • 通过 "{register} 前缀指定寄存器，不指定默认用无名寄存器
    • 有名寄存器 a-z
    • 复制专用寄存器 "0 使用 y 复制的文本同时会被拷贝到复制寄存器 0
    • 系统剪贴板 "+ 可以在复制前加上 "+ 复制到系统剪贴板
    • 其他寄存器 "% 当前文件名， “. 上次插入的文本

撤销

命令 作用
u undo

分屏

命令 作用
:vs vertical split
:sp horizontal split

文本选择

命令 作用
v enter visual mode, then use direction key to select chars
V select the whole line
ctrl+v select a bolck using direction key

插入模式中删除

命令 作用
ctrl+h delete previous char
ctrl+w delete chars from the start of the word to the cursor (when the cursor is at the tail of the word, delete the whole word)
ctrl+u delete chars from the start of the line to the cursor (when the cursor is at the tail of the line, delete the whole line)

插入模式中粘贴

命令 作用
ctrl+v paste
